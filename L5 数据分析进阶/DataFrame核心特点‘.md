**DataFrame** 是一种广泛应用于数据分析和数据科学中的**二维表格型数据结构**。你可以把它想象成一个 Excel 表格或 SQL 数据库中的表。

它是 Python 数据分析库（如 **Pandas**）、Spark 或 R 语言中最核心的对象。

### 核心特点：

1. **行与列**：数据按行和列排列。每一行代表一条记录，每一列代表一个字段（属性）。
2. **异质性**：同一列中的数据类型必须相同，但不同列之间可以拥有不同的数据类型（例如，第一列是字符串，第二列是整数，第三列是浮点数）。
3. **索引（Index）**：每一行都有一个标签（索引），每一列都有一个列名。这使得数据的查询、切片和修改非常方便。
4. **大小可变**：你可以轻松地增加新列、删除旧列或合并多个 DataFrame。

### 简单示例：

如果用代码（Pandas）创建一个 DataFrame，可以先用键值对（字典）的方式构造原始数据：

```python
data = {
    "姓名 (Name)": ["张三", "李四", "王五"],
    "年龄 (Age)": [25, 30, 22],
    "城市 (City)": ["北京", "上海", "广州"]
}
df = pd.DataFrame(data)
```

这个 DataFrame 显示如下：

|       | 姓名 (Name) | 年龄 (Age) | 城市 (City) |
| :---- | :---------- | :--------- | :---------- |
| **0** | 张三        | 25         | 北京        |
| **1** | 李四        | 30         | 上海        |
| **2** | 王五        | 22         | 广州        |

### 常见用途：

- **数据清洗**：处理缺失值、重复数据。
- **统计分析**：计算平均值、中位数、相关性等。
- **数据转换**：对数据进行排序、筛选、分组（Group By）或透视。
- **机器学习预处理**：作为输入特征提供给算法模型。

**总结：** DataFrame 是处理结构化数据最强大、最直观的工具，是进行数据挖掘和分析的基础。
