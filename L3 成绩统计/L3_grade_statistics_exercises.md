## Exercise 2: 气象数据分箱统计 ⭐⭐⭐

这次换一个完全不同的数据集：气象观测记录。

文件 `climate_data_2017.csv` 是一个 CSV 文件，每一行用逗号 `,` 分隔，有 13 列。我们只关心其中三列：

| 列索引 | 列名 | 含义 |
|--------|------|------|
| 9 | wdir | 风向（如 N, NE, E, SE, S, SW, W, NW） |
| 11 | temperature | 温度（小数） |
| 12 | humidity | 湿度（小数） |

你的任务是：按**风向**和**湿度分箱**对数据进行分组，然后求每组的**平均温度**。

湿度分箱的意思是：用 `humidity // 20` 把湿度值归入不同的区间。比如湿度 45.0 → `45.0 // 20 = 2.0`，湿度 72.0 → `72.0 // 20 = 3.0`。

**要求：**

1. 读取 `climate_data_2017.csv`
2. 跳过第一行表头
3. 每一行用 `split(',')` 拆开
4. 取出 `values[9]`（风向）、`values[11]`（温度）、`values[12]`（湿度）
5. 温度和湿度要转成 `float`
6. 计算湿度分箱：`humidity // 20`
7. 用 `(wdir, humidity_bin)` 这个元组当字典的 key
8. 每个 key 对应一个温度列表，把同组的温度都存进去
9. 最后按 key 排序输出，每行格式：`key : 平均温度`
10. 平均温度保留 1 位小数，用 `round(..., 1)`
